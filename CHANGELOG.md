# Changelog

All notable changes to this project will be documented in this file.

## 1.2.0 - 2026-03-03

- Added dual Craft CMS compatibility support (`^4.0 || ^5.0`) in Composer constraints.
- Updated package metadata and documentation to reflect Craft 4/5 support.

## 1.1.0 - 2026-02-27

- Removed organization-specific branded default copy from plugin defaults.
- Replaced branded default text values with neutral locale defaults and neutral fallback text.
- Updated CP/frontend UI labels to avoid explicit locale code display.
- Updated Composer metadata for Craft Plugin Store readiness.
- Added MIT license and package documentation files.
